Todos sabemos que en los ordenadores viene integrada una BIOS. Pero, ¿sabemos realmente que es una BIOS?
BIOS proviene de las siglas "Basic Input Output System" (Sistema basico de entrada y salida).
Basicamente, es el primer sistema operativo que se pone en marcha cuya funcion es poner el hardware en funcionamiento antes del sistema operativo principal.

UEFI (Unified Extensible Firmware Interface) es el firmware sucesor de las BIOS convencionales. UEFI realiza todas las funciones de BIOS y otras funciones adicionales, como por ejemplo una interfaz gráfica, un sistema de inicio seguro llamado Secure Boot, entre otras.


Se encuentra en la memoria ROM, es decir, que es una memoria que no se puede borrar o que es una memoria no volátil. Sobre el cual se encuentran asignados las primeras direcciones dememoria a las que el procesador debe de acceder.
La UEFI se encarga de funciones a muy bajo nivel, esto quiere decir que su función es más lógica que física.

ESTAS SON LAS PRINCIPALES FUNCIONES DE LA UEFI:

  1. iniciar los componentes de hardware y lanzar el sistema operativo de un ordenador cuando lo encendemos.
  2. Manejo más fácil con ayuda del ratón y de la interfaz gráfica.
  3. Gestor de arranque integrado, que administra diferentes gestores de arranque de diferentes sistemas operativos.
  4. Herramienta de línea de comandos para el diagnóstico y la búsqueda de errores (UEFI Shell).
  5. Conectividad de red incluso sin el sistema operativo no está activo.
  6. Mayor seguridad gracias al modo Secure Boot.
  7. Y todas las opciones que ofrece una BIOS clásica.

Todas estas funciones se pueden hacer de manera directa desde la UEFI, sin necesidad de acceder al sistema operativo. Lo cual es sumamente útil si no tenemos un sistema operativo instalado en el PC y necesitamos realizar una de las gestiones que se encuentran en la lista de arriba.
El modo Secure Boot se introdujo para hacer el proceso más seguro. Cada componente de software se verifica antes de iniciarse utilizando firmas criptográficas, introducidas en la base de datos de firmas del firmware UEFI. Si alguno de estos componentes está manipulado o no posee firma o una clave válida, no pasará el test de seguridad y el sistema interrumpirá el arranque.

Existen tantas maneras de acceder a la BIOS como las diferentes UEFI en sí. Aquí mostramos algunas marcas y la combinación de teclas para su acceso.

  • HP: F10
  • Lenovo: F2 o Fn + F2 en los portátiles, F1 en sus PC de escritorio y Enter y luego F1 en el caso de que estés utilizando un Thinkpad
  • Gigabyte / Aorus: F2 o SUPR
  • MSI: SUPR
  • ASRock: F2 o SUPR.
  • ASUS: F2 para todos los PCs, F2 o SUPR para las placas base.
  • Acer: F2 o SUPR.
  • Dell: F2 o F12


Windows 10 tambien permite ingresar a la UEFI desde el pripo sistema operativo. Eso sí, de reiniciar el sistema no nos libra nadie.

Accedemos al menú de configuración, y clicamos en Actualización y seguridad.

Clicamos en Recuperación, sutiado en el menú vertical en la parte izquierda. Y clicamos en Reiniciar ahora, en el apartado Inicio Avanzado.

Una vez reiniciado el sistema, nos aparecerá el siguiente menú, en el que seleccionaremos Solicionar problemas.

En el siguiente paso, seleccionamos Opciones avanzadas.

Por último en la siguiente ventana, selecionamos Configuración de firmware UEFI.

Y reiniciamos.

Al reiniciar, nos saldrá el siguiente menú, y selecionamos mediante las teclas de función la acción deseada.

Empezaremos a securizar nuetra UEFI en el siguiente orden:

  • Contraseña de usuario o Power-On.
  • Constraseña de administrador.
  • Deshabilitar arranques externos.
  • Establecer un orden de arranque.
  • Otras opciones de seguridad.

CONTRASEÑA DE USUARIO O POWER-ON.

Para evitar cualquier intrusión en nuestro pc, el primer paso es habilitar una contraseña en el inicio del arranque del sistema. Esta opción se habilita en la pestaña de SEGURIDAD, en la opción CONTRASEÑA DE ARRANQUE.

Introducimos la contraseña. Las contraseñas deben de cumplir los siguientes requisitos de complejidad como mínimo:

  • 8 caracteres
  • Mayúsculas
  • Minúsculas
  • Dígitos de base 10
  • Carateres no alfanuméricos

Una vez que reiniciemos el equipo, nos pedirá la contraseña que hemos configurado en el paso anterior.

CONTRASEÑA DE ADMINISTRADOR.

Es una configuración de las más importantes. Se trata de restringir el acceso a la configuracion de la UEFI. En la misma pestaña de SEGURIDAD que en el paso anterior, y la opción CONTRASEÑA DE ADMINISTRADOR se restringe esta entrada.

Una vez habilitada esta opción, nos pedirá que introdizcamos una contraseña. La contraseña más favorable es siempre la que cumpla los mínimos de los requisitos de complejidad vistos en el paso anterior.

Al igual que en la contraseña de POWER-ON, al reiniciar el equipo nos saldrá, en este caso, la misma ventana. Pidiendo que introduzacamos la contraseña.

DESHABILITAR ARRANQUES EXTERNOS.

El principal motivo por el que se realiza esta acción no es otra que la seguridad del pc, ya que si se tiene habilitado este tipo de arranque se podrian realziar la instalación de un sistema operativo, usar un software para recuperar contraseñas, instalar programas no deseados o el uso de CDs con software con el fin de obtener archivos o eliminarlos.
El primer paso es el acceso a la UEFI como en los pasos anteriores. En nuestra UEFI nos desplazaremos hasta el menú Configuración del sistema.

Para deshabilitar la posibildad de un arranque externo, nos adentramos en el submenú Opciones de arranque, donde desactivaremos TODO excepto menos el Arranque seguro o Secure boot. Esta opción se explicará más avanzada esta guía.

NOTA: CON ESTA CONFIGURACIÓN YA TENDRÍAMOS INHABILITADA CUALQUIER ARRANQUE QUE NO SEA EL DEL PROPIO SISTEMA OPERATIVO.

ORDEN DE ARRANQUE DEL SISTEMA.

Cuando arrancamos nuestro ordenador, lo primero que carga es el firmware UEFI o el BIOS para posteriormente iniciarse el sistema operativo. Pero enfocando esta guía a la seguridad, no es recomendable que el arranque por dispositivos externos o USBs esten activados.
El cambio de orden de arranque se realiza en la opción Configuración del sistema, en el submenú Opciones de arranque. Pondremos como primera opción Administrador de arranque del SO. Esto quiere decir que "mirará" donde está el boot Manager del sistema operativo.

OTRAS OPCIONES DE SEGURIDAD.

Secure Boot o Arranque Seguro impide la ejecución de cualquier software no fimardo o certificado en el arranque del sistema. Con esto se intenta evitar la carga de malware o aplicaciones que no sean de fiar cuando inicias el ordenador, pero tambien afecta a sistemas opertivos como las distribuciones GNU/Linux.
Secure Boot o Arranque Seguro se habilita en el menú Opciones del sistema en la opción Arranque Seguro.
