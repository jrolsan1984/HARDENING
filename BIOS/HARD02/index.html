Sin la BIOS (Basic Input-Ouput System) nada funcionaria en nuestro PC. En ocasiones puede mejorar el rendimiento del PC o ampliar sus capacidades, modificando la configuración.
La BIOS es un programa que ya está instalado en el ordenador, en concreto en su placa base. Lo primero que hace es un chequeo de todos los componentes de hardware. Si encuentra algún fallo, se encarga de avisarte a través de un mensaje en pantalla, o mediante los "tradicionales" pitidos de alarma.

La BIOS está grabada en un pequeño chip de memoria situado en la placa base.En los ordenadores antiguos (los de hace tan sólo unos cuantos años) no era posible borrar esta información, sino tan sólo leerla. Por eso recibe también el nombre de ROM (ReadOnly Memory, o Memoria de Sólo Lectura), y a menudo aparece en libros y manuales como ROM-BIOS.
Ya hace algún tiempo que se puede modificar la BIOS. Para ello se emplea el llamado "Setup" de la BIOS. Como los parámetros de configuración básica se escriben en una memoria CMOS de la placa base, a esta utilidad se la conoce como CMOS Setup Utility, o "Programa de ayuda de Configuración CMOS". La BIOS almacena también datos acerca de la configuración en una parte especial del chip de memoria BIOS, el NVRAM .

ESTAS SON LAS PRINCIPALES FUNCIONES DEL BIOS:

  1. Es el encargado de ejecutar los procesos POST (Power On Self Test), estos verifican si la memoria RAM, los periféricos y demás componentes de hardware funcionan correctamente antes de comenzar a cargar el sistema operativo.
  2. Otra de las funciones principales es la de buscar y cargar el sistema operativo, esta se llama BOOTSTRAP LOADER o BOOTLOADER. Cuando encuentra el sector de arranque en el dispositivo de almacenamiento comienza la carga del sistema operativo instalado en el dispositivo de almacenamiento primario. Inmediatamente se le pasa el control de las funciones principales al sistema operativo.
  3. A través de Interfaz Avanzada de Configuración y Energía(ACPI) se controla el funcionamiento del BIOS. Además, permite modificar los ajustes de hardware y gestionar el consumo de energía.
  4. El controlador BIOS se encarga de darle al usuario los controles básicos de funcionamiento de hardware y periféricos como el teclado.
  5. Comprueba todos los componentes de hardware conectados a la placa base y los dispositivos de almacenamiento en el orden definido por el usuario.

EL acceso a la configuración es a través de la Interfaz Avanzada de Configuración y Energía (ACPI). Para acceder a esta interfaz deberá presionar varias veces una de las teclas: F1, F2, F8, F10, ESC o DEL dependiendo del fabricante de la placa base. En caso de que aparezca un menú deberá seleccionar Entrar a la configuración del BIOS.

Se debe tener en cuenta que la configuración se guarda en la memoria CMOS, en caso de que note que la configuración del BIOS no se mantiene cuando desconecta el ordenador del fluido eléctrico deberá cambiar la pila CMOS.

COMO RESTABLECER LA CONFIGURACIÓN DEL BIOS.

Si notamos que alguna configuración realizada en el BIOS afecta el correcto funcionamiento de nuestro ordenador, se puede reiniciar la configuración del BIOS y regresar a sus ajustes predeterminados. Esta operación se puede hacer de dos formas:

  1. Seleccionando la opción Configuración predeterminada de fábrica en la interfaz de configuración del BIOS.
  2. Extraer la batería CMOS de la placa base durante unos minutos. La batería CMOS es la que permite que la configuración del BIOS guardada en la memoria CMOS no se pierda.
  3. Usar el jumper o botón Clear CMOS o Reset CMOS, previo apagado del ordenador, cambiar el par de pines unidos (en caso de que sean 3) por la pieza de plástico, luego comprobar los cambios y volver a poner la pieza en los pines iniciales. En caso de que sean 2 pines quitar a pieza plástica.

Empezaremos a securizar nuestra BIOS en el siguiente orden:

  • Contraseña de usuario o Power-On.
  • Constraseña de administrador.
  • Deshabilitar arranques externos.
  • Establecer un orden de arranque.
  • Otras opciones de seguridad.

CONTRASEÑA DE USUARIO O POWER-ON.

Para evitar cualquier intrusión en nuestro pc, el primer paso es habilitar una contraseña en el inicio del arranque del sistema. Esta opción se habilita en la pestaña de SECURITY, en la opción USER PASSWORD.

Introducimos la contraseña. Las contraseñas deben de cumplir los siguientes requisitos de complejidad como mínimo:

  • 8 caracteres
  • Mayúsculas
  • Minúsculas
  • Dígitos de base 10
  • Carateres no alfanuméricos

CONTRASEÑA DE ADMINISTRADOR.

Es una configuración de las más importantes. Se trata de restringir el acceso a la configuración del BIOS. En la misma pestaña de SECURITY que en el paso anterior, y la opción ADMINISTRATOR PASSWORD se restringe esta entrada.

DESHABILITAR ARRANQUES EXTERNOS.

El principal motivo por el que se realiza esta acción no es otra que la seguridad del pc, ya que si se tiene habilitado este tipo de arranque se podrian realziar la instalación de un sistema operativo, usar un software para recuperar contraseñas, instalar programas no deseados o el uso de CDs con software con el fin de obtener archivos o eliminarlos.
El primer paso es el acceso al BIOS como en los pasos anteriores. En nuestra BIOS nos desplazaremos hasta el menú BOOT y selecionamos Disable en todas las opciones marcadas.

  1. USB Boot deshabilita el inicio de cualquier arranque de sistema por USB.
  2. PXE Boot to LAN. Esta opción permite arrancar la tarjeta de red en remoto. Por seguridad, es conveniente tenerla desactivada ya que es una via de entrada por parte de un atacante.
  3. IPV4 PXE First. Esta opción permite arrancar la tarjeta de red mediante el protocolo IPv4. Si tenemos la opción PXE Boot to LANdesactivada, esta última desaparece de las opciones.

OTRAS OPCIONES DE SEGURIDAD.

Secure Boot o Arranque Seguro impide la ejecución de cualquier software no fimardo o certificado en el arranque del sistema. Con esto se intenta evitar la carga de malware o aplicaciones que no sean de fiar cuando inicias el ordenador, pero tambien afecta a sistemas opertivos como las distribuciones GNU/Linux.
Secure Boot o Arranque Seguro se habilita en el menú SECURITY en la opción Secure Boot.

Otra opción de seguridad que ofrece este BIOS es la posibilidad de añadir una password al disco donde tenemos alojado el sistema operativo. Esta opción se encuentra en la pestaña de SECURITY, en la opción Set Hard Disk Password.
