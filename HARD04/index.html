El sistema operativo Windows 10 nos ofrece distintas opciones a la hora de iniciar sesión. Es posible elegir un PIN, una contraseña, a través de datos biométricos, usar una imagen como contraseña... Para los usuarios "de a pie" lo más común es establecer una contraseña o un PIN para proteger la entrada a nuestro dispositivo.
Si esa contraseña o PIN se supiera por cualquier motivo, cualquier otro usuario podrian cambiarnos la contraseña e imposibilitarnos el acceso a nuestro dispositivo. Lo más correcto sería impedir que nadie pudiera cambiar la contraseña de usuario.

ELECCIÓN DE TIPO DE CONTRASEÑA.

Cómo acabamos de explicar, el sistema operativo Windows nos ofrece distintas maneras de proteger el acceso a nuestro dispositivo. Para elegir que tipo de bloqueo queremos instaurar, nos iremos desplazando en el menú de INICIO hasta llegar al sitio de Configuración > Cuentas > Opciones de inicio de sesión.

DESDE LÍNEA DE COMANDOS.

Un método rápido esa través de línea de comandos. Esta forma de impedir cambiar la contraseña de un usuario está disponible en todas las ediciones de Windows 10. Para conseguier este propósito seguimos los siguientes pasos:

  1. Abrimos una ventana del símbolo del sistema con permisos de administrador.
  2. Escribimos el comando net user "nombreusuario" /PasswordChg:No. Donde nombreusuario debe sustituirse por el nombre de usuario del sistema al que queremos bloquear el cambio de contraseña. Debemos escribir el nombre exacto y sin comillas.
  3. Finalizamos pulsando Enter para ejecutar el comando.

DESDE POLÍTICAS LOCALES DEL SISTEMA.

Si contamos con la edición Pro, Enterprise o Education de Windows 10, entonces también podemos impedir que un usuario pueda cambiar la contraseña de su cuenta a través de la directiva de grupo del sistema. En esta ocasión, los pasos a seguir son:

  1. Abrimos una ventana Ejecutar, Win+R.
  2. Escribimos lusrmgr.msc y pulsamos Enter.
  3. Seleccionamos la carpeta de Usuarios dentro de Usuario y grupos locales.
  4. Buscamos el nombre de usuario al que queremos bloquear el cambio de contraseña.
  5. Hacemos clic sobre él con el botón derecho del ratón y seleccionamos la opción Propiedades.
  6. Esto nos abrirá la ventana de propiedades del usuario en cuestión, donde seleccionamos la pestaña General.
  7. Marcamos la casilla El usuario no puede cambiar la contraseña. Aplicamos y para que se guarden los cambios y pulsamos Aceptar.

Con estos sencillos pasos evitamos que cualquier otro usuario nos cambie la contraseña y evitamos cualquier tipo de problema.
